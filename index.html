Intelligent Cloud Operations Agent

LangChainPythonTerraformFastAPIGCP

Problem Statement

Cloud operations teams often rely on manual queries and console navigation for routine diagnostics. This project aimed to create an AI agent to automate these tasks using natural language, providing instant Infrastructure-as-Code (IaC) awareness.

Architecture

Technical Decisions & Outcomes

  • Architectural Choice: A self-hosted Hugging Face model on a GCE VM was chosen over proprietary APIs (like OpenAI) to ensure data privacy and eliminate API costs. The trade-off was increased management overhead, acceptable for this proof-of-concept.
  • Technical Challenge: Securely parsing the Terraform state file required creating a dedicated GCP Service Account with only the `storage.objects.get` permission on the specific `.tfstate` object, adhering to the principle of least privilege.
  • Quantifiable Outcome: Automated 5 core operational checks, reducing the time for routine diagnostics from an average of 5-10 minutes manually to **under 15 seconds** via a natural language query.
